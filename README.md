# Passkeys Demo

This is a simple demo of [Passkeys Authentication](https://passkeys.dev/) using
[SimpleWebAuthn](https://simplewebauthn.dev/) and [Deno](https://deno.com/runtime). Usernameless authentication is
supported.

## Requirements

- Deno `1.32` or higher
